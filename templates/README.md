# Update templates

Word templates for the four Updates, generated by `generators/build_templates.js`.

```
npm install docx
node generators/build_templates.js
```

This produces:
- First-Update-Template.docx
- Second-Update-Template.docx
- Pre-Final-Update-Template.docx
- Final-Update-Template.docx
